# 25L-2560-Chess
PF Chess Project
SFML Chess

A simple 2D chess game in C++ using SFML. Supports piece movement, capturing, check, and checkmate.

Features

2-player local chess

Legal moves for all pieces

Pawn promotion to Queen

Score tracking for captured pieces

Requirements

C++17 compiler

SFML library

Chess piece images (images/) and font (fonts/arial.ttf)
